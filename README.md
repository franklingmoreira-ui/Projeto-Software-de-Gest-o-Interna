Sistema robusto de gestão interna com metodologia Kanban, desenvolvido para otimizar o fluxo de trabalho. Este projeto faz parte do meu portfólio de Engenharia de Software e demonstra a integração de tecnologias modernas para soluções empresariais.

## 🛠 Tecnologias Utilizadas

- **Backend:** FastAPI (Python) com SQLAlchemy ORM.
- **Frontend:** React.js + Vite.
- **Banco de Dados:** MySQL 8.0 (Migrado de SQLite para alta escalabilidade).
- **Infraestrutura:** Docker & Docker Compose para orquestração de containers.

## 📋 Funcionalidades

- [x] Autenticação de usuários (Admin/User).
- [x] Painel Kanban dinâmico para lançamento e gestão de demandas.
- [x] Persistência de dados em ambiente relacional MySQL.
- [x] Interface responsiva e moderna.
- [ ] Integração com AWS (Em progresso).
